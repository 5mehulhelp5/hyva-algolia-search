# hyva-algolia-search
Compat module for Algolia Search on Magento 2 using Hyv√§ Themes. This uses blackbird/external-ressource-loader to dynamically load all the Algolia ressources. 

## Setup

Get the package

Composer Package:
```composer require blackbird/module-hyva-algolia-search```

Zip Package:
Unzip the package in app/code/Blackbird/HyvaAlgoliaSearch, from the root of your Magento instance.
